# OpenFuncDB 项目规划

## 项目定位

为 AI 智能体提供安全合规的结构化技术数据 API，覆盖 Linux 命令、Python 函数、Git 操作、Shell 脚本等。

## 目标

- 为 AI 智能体提供可在线调用的安全函数/命令知识库
- 所有数据经过安全审核，标记安全等级
- 支持多类型技术数据（Python/Linux/Git/Shell）
- 开源可协作，达到 GitHub 高星项目水准

## 技术路线

| 组件 | 选型 | 说明 |
|------|------|------|
| Web 框架 | FastAPI | 高性能异步框架，自动生成 API 文档 |
| ORM | SQLAlchemy 2.0 | Python 最成熟的 ORM |
| 数据库 | MySQL 8.0 | 稳定可靠的关系型数据库 |
| 迁移工具 | Alembic | 数据库版本管理 |
| 鉴权 | JWT + OAuth2 | 无状态令牌认证 |
| 限流 | SlowAPI | 接口访问频率控制 |
| 部署 | Uvicorn + Docker | 轻量高性能部署方案 |
