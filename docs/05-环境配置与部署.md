# OpenFuncDB 环境配置与部署指南

## 开发环境配置

### 前提条件

- Python 3.10+
- MySQL 8.0+
- Anaconda（推荐）

### 步骤

1. **克隆项目**

```bash
git clone https://github.com/1008wjh/OpenFuncDB.git
cd OpenFuncDB
```

2. **创建虚拟环境**

```bash
conda create -n openfunc python=3.12
conda activate openfunc
```

3. **安装依赖**

```bash
pip install -r requirements.txt
```

如使用国内镜像加速：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

4. **配置数据库**

编辑 `.env` 文件：

```ini
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=root
DB_NAME=openfunc_db

JWT_SECRET_KEY=your_secret_key
JWT_ACCESS_TOKEN_EXPIRE_MINUTES=30

BACKEND_CORS_ORIGINS=http://localhost:5173,http://127.0.0.1:5173
```

5. **初始化数据库**

```bash
# 创建数据库
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS openfunc_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 执行迁移
python -m alembic upgrade head
```

6. **启动服务**

```bash
python main.py
```

访问 http://localhost:8000/docs 查看 API 文档。

7. **初始化管理员**

```bash
curl -X POST http://localhost:8000/api/v1/auth/init-admin
```

默认账号：`admin` / `admin123`

## 常用命令速查

```bash
# 启动开发服务器
python main.py

# 数据库迁移
python -m alembic revision --autogenerate -m "描述"
python -m alembic upgrade head

# 查看数据库当前版本
python -m alembic current

# 查看迁移历史
python -m alembic history
```

## 环境变量说明

| 变量 | 默认值 | 说明 |
|------|--------|------|
| DB_HOST | localhost | 数据库地址 |
| DB_PORT | 3306 | 数据库端口 |
| DB_USER | root | 数据库用户 |
| DB_PASSWORD | root | 数据库密码 |
| DB_NAME | openfunc_db | 数据库名 |
| JWT_SECRET_KEY | (内置开发密钥) | JWT 签名密钥（生产环境必须修改） |
| JWT_ALGORITHM | HS256 | 加密算法 |
| JWT_ACCESS_TOKEN_EXPIRE_MINUTES | 30 | Token 有效期（分钟） |
| BACKEND_CORS_ORIGINS | localhost:5173 | 跨域允许源 |
| PROJECT_NAME | OpenFuncDB API | 项目名称 |

> **安全提醒**：生产环境必须修改 `JWT_SECRET_KEY`，建议使用强随机字符串。
